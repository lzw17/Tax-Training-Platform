## 🐛 常见问题排查

### 问题 1：登录后空白页
**解决方案：**
1. 打开浏览器控制台（F12）
2. 查看 Console 是否有错误
3. 查看 Network 面板，检查 `/api/auth/login` 和 `/api/users/profile` 请求
4. 清除浏览器缓存和 localStorage

### 问题 2：API 请求失败
**解决方案：**
1. 检查后端服务是否正常运行：`curl http://localhost:3001/health`
2. 检查数据库连接是否正常
3. 查看后端日志：`tail -f logs/backend.log`

### 问题 3：页面显示"获取列表失败"
**可能原因：**
1. 后端 API 未实现或有错误
2. 数据库表不存在
3. 权限问题

**解决方案：**
1. 检查后端日志
2. 确认数据库表已创建
3. 检查 API 路由是否正确

### 问题 4：创建/编辑操作失败
**可能原因：**
1. 表单验证失败
2. 后端 API 返回错误
3. 数据库约束冲突

**解决方案：**
1. 查看页面错误提示
2. 打开浏览器控制台查看详细错误
3. 检查后端日志
