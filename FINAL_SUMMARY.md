## 📝 使用指南

### 启动项目
```bash
# 1. 激活环境
conda activate tax-platform

# 2. 启动开发服务
./scripts/start-dev.sh

# 3. 访问系统
# 前端：http://localhost:3000
# 后端：http://localhost:3001
```

### 默认账户
- **管理员：** admin / password
- **教师：** teacher01 / 123456（需先创建）
- **学生：** student01 / 123456（需先创建）

### 页面路由
```
/login                          - 登录页面
/dashboard                      - 仪表板
/dashboard/users                - 用户管理
/dashboard/courses              - 课程管理
/dashboard/questions            - 试题管理
/dashboard/exams                - 考试管理
/dashboard/grades               - 成绩管理
/dashboard/exam/:examId         - 考试答题
/dashboard/profile              - 个人资料
```
