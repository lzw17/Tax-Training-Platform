## 🚀 快速启动

### 开发环境
```bash
# 1. 初始化数据库
./scripts/setup-db.sh

# 2. 启动开发环境
./scripts/start-dev.sh

# 3. 访问应用
# 前端: http://localhost:3000
# 后端: http://localhost:3001
# 默认账户: admin / password
```

### 生产部署
```bash
cd deploy
cp .env.example .env
# 编辑环境变量
docker-compose up -d
```
